About ImproveMint.

A self improvement platform built with Python (Flask)

ImproveMint was my first ever web-development project.

It taught me the basics of HTML, CSS, JavaScript and MySQL, as well as how basic web-development works. (HTTP requests, sessions, databases, etc.)

Since the first version of ImproveMint, my knowledge of web-development has grown exponentially, causing me to return to this project and refactor the code to make it modular, as well as improve the frontend design and the backend code.
